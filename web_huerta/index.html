Proyecto Monitoreo y control de un cultivo hidropónico NFT

Hydrotech

Objetivo general.

Implementar tecnologías IoT (Internet de las cosas) en un sistema de monitoreo y control de un cultivo hidropónico NFT haciendo uso de  herramientas de hardware y software de uso libre. 

Descripción del proyecto.

Este proyecto es una investigación en la que se estudian tecnologías de uso libre en proyectos de automatización de centros agrícolas para luego implementar un sistema de monitoreo y control de las variables fundamentales en un cultivo hidropónico tipo NFT, las cuales son: Humedad, temperatura, luminosidad, PH, conductividad. Este proyecto va dirigido inicialmente a comunidades huerteras y agricolas que estén interesados en tener el control de sus cultivos para tener un mejor resultado.

Objetivo del proyecto(utilidad)

Con el proyecto lo que se busca es, inicalmente el incentivar a las comunidades a cultivar su propio alimento y que con este proyecto pueda tener un control de su sistema implementado. Además, los agricultores o huerteros que ya tengan experiencia pueden implementar el proyecto para facilitar sus tareas y logren optimizar sus sistemas.
